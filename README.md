# AGMS Microservices Platform

## Project Overview

The **Automated Greenhouse Management System (AGMS)** is a microservices-based platform for managing greenhouse operations.  
It provides modular services for monitoring environmental conditions, controlling devices, and automating processes.

### Core Services

1. **API Gateway** – Routes requests to appropriate microservices.  
2. **Discovery Server** – Handles service discovery for microservices.  
3. **Zone Service** – Manages greenhouse zones.  
4. **Crop Service** – Tracks crops and related information.  
5. **Sensor Service** – Collects and stores sensor data (temperature, humidity, etc.).  
6. **Automation Service** – Controls devices like fans and irrigation based on rules.  
7. **Config Server** – Centralized configuration for all services.
